# Retail Sales Dashboard (Power BI)

This project demonstrates an end-to-end Power BI workflow including:
- Data cleaning and transformation using Power Query
- Data modeling with fact and dimension tables
- DAX measures for sales and profit analysis
- Interactive dashboard creation

## Dashboard Overview
The dashboard provides insights into:
- Total Sales, Profit, Profit Margin, and Orders
- Monthly sales trends
- Profit by category
- Regional sales performance
- Top 10 customers by sales

## Files in this Repository
- Sales_Dashboard(Report).pbix – Power BI report file
- Sales_Dashboard.pdf – Exported dashboard view
- Globalsuperstore dataset.xlsx- Dataset Excel File

## Tools Used
- Power BI Desktop
- Power BI Service
- Power Query
- DAX
